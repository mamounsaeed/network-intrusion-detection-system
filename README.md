# Network Intrusion Detection System (NIDS)

Real-time intrusion detection system using Deep Learning.

## Features
- Real-time attack detection
- Flask Dashboard
- ANN Deep Learning Model
- Detection of:
  - DoS
  - DDoS
  - Port Scan
  - Brute Force
  - Bot
  - Web Attack
  - Infiltration

## Technologies
Python, TensorFlow, Keras, Flask, Pandas, NumPy, Scikit-learn
